## 关于 SpawnPlus

## 简介
SpawnPlus是一款可以设置玩家重生点优先级的插件  
重生点优先级有3个, 最优先的是local(个人), 其次则是group(小组), 最后则是default(默认/全局)。  
如果玩家已经设置了个人出生点, 那么他们在登入传送/死亡/传送时前往该传送点, 然而如果玩家没有设置个人传送点, 插件将会为其选择小组传送点用于传送。否则, 如果玩家没有个人或者小组传送点, 那么默认的全局传送点将会被选择用于传送。
